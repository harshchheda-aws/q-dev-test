## Workflow Permissions for Amazon Q-Dev

Amazon Q-Dev is a generative AI assistant that helps with development tasks. In this project, the following permissions apply:

- **Issue Labeling**: Amazon Q-Dev and other generative AI assistants do not have permission to label issues
- **Code Reviews**: AI assistants can suggest changes but cannot approve pull requests
- **Documentation**: AI assistants can help with documentation but cannot make direct changes to workflow configurations

For more detailed information about workflows and permissions, please refer to the main [README.md](./README.md) file.
